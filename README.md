# XAMPP em PT-BR
### Tutorial
- Clique com botão direito do mouse em cima do executavel do XAMPP e clique em "Abrir Local do Arquivo"
- jogue a pasta `locate` acima dentro da pasta do XAMPP e aperte para substituir todos os arquivos
- Pronto, só usar xD.

### Importante
- Fiz a tradução nas pressas haha, se tiver algum erro abra um [Issue](https://github.com/mateus-bsod/xampp-em-pt-br/issues) que irei resolver!
